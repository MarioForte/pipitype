# Ту-Ту-Тайп! (tututype)

Клавиатурный тренажёр для набора текста на русском и английском языках. Простое и минималистичное веб-приложение, работающее полностью на стороне клиента.

## Возможности

- Печать на русском и английском языках
- Метрики: CPM (символы в минуту), точность, время
- Подсветка текущего, правильного и неправильного символов
- Исправление ошибок с помощью Backspace
- Показ предыдущего результата
- Сохранение истории результатов в localStorage
- Страница статистики с графиком прогресса CPM
- Оценка скорости печати по возрасту

## Технологии

Чистый HTML, CSS и JavaScript (vanilla JS). Никаких зависимостей, сборщиков или серверной части.
